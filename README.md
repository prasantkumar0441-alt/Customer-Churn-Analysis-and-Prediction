# Customer Churn Analysis and Prediction

## Project Overview

This project focuses on analyzing customer churn in a telecommunications company and developing machine learning models to identify customers who are likely to leave the company.

The project includes data preprocessing, exploratory data analysis (EDA), customer segmentation, churn prediction, model evaluation, and business recommendations.

---

## Objectives

* Analyze customer churn behavior
* Identify key factors influencing churn
* Segment customers based on tenure and monthly charges
* Build predictive machine learning models
* Evaluate model performance
* Provide actionable business recommendations

---

## Dataset Information

Dataset: Telco Customer Churn Dataset

Total Records: 7043

Features: 21

Target Variable: Churn (Yes / No)

| --------- | ------ |

| Recall    | 60.05% |
| F1 Score  | 64.00% |

---

## Key Findings

* Month-to-Month contract customers exhibited the highest churn rate.
* Electronic Check users showed significantly higher churn.
* New customers were more likely to churn.
* Loyal customers demonstrated strong retention.
* Fiber Optic users were identified as a major churn-risk segment.
